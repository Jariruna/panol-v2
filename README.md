# Sistema de Gestión de Pañol
Aplicación web para control de inventario de herramientas.

## Características
- **Persistencia**: Uso de `localStorage`.
- **Interactivo**: Cambio de estado de herramientas con un clic.
- **Validación**: Control de duplicados integrado.
- **Diseño**: Interfaz industrial responsive.

## Tecnologías
- JavaScript (Módulos ES6)
- CSS3 (Flexbox, Variables)
